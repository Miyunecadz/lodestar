---
id: protect-dbmate-schema
title: Protect the dbmate-generated schema
category: database
severity: block
recommended: true
stacks: [node-dbmate]
event: file
pattern: 'db/schema\.sql$'
emits: hookify
---

`db/schema.sql` is generated by dbmate; hand-edits are overwritten on the next migrate and cause drift between the file and the real database. Change the schema by writing a new migration instead — `schema.sql` updates itself when the migration runs.
