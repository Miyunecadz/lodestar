# Frontend standards (REPO)

Conventions live in the docs, not here. Read **`docs/REPO/conventions.md`** and **`docs/REPO/architecture/`** before editing.

**Key reminders:**

- **Data:** Apollo Client cache patterns (fetch policies, cache updates, normalization) are documented in `docs/REPO/`.
- **Grids:** ag-grid patterns are documented there too — follow them rather than inventing new grid wiring.
- Components are React on **CRACO**, styled with Tailwind.

Details and the actual component/module layout are in `docs/REPO/` — go there.
